# s/w 프로젝트 주제 정리 

1. 무한런 2D 게임 
2. 키넥트 1.8v과 유니티 2018 구버전을 사용(다른 버전 거의 안됨, 유니티 6, 2021, 2022 호환 x)
3. 쿠키런처럼 플레이어의 움직임은 점프와 슬라이드만 가능
4. 플러그인 연동은 dll 플러그인보다 무료버전의 에셋스토어의 플러그인을 임포트(dll파일 임포트해도 안됨, 버전이 달라서 안됨)
5. 임포트를 마치고 나서 하이라키 창에 빈 게임 오브젝트를 생성해서 KinectManager 스크립트 할당해서 그 기능을 참조해서 사용함

6. 플레이어가 갖는 컴포넌트 리지드바디2D(2D게임용 물리 작용), 박스 콜라이더(충돌 처리 및 바닥 인식 처리), playerController 스크립트
- 플레이어는 2단점프를 못하게 바닥에서만 점프(플레이어 오브젝트와 땅 오브젝트에 각각 player 태그와 ground 태그 추가)
- PlayerController 스크립트에는 KinectManager API를 통해 스켈레톤 데이터 가져와서 사용
- 점프를 인식해서 플레이어가 점프, 슬라이드를 인식해서 플레이어가 슬라이드, 인식을 잘못함
--------------------------- 키넥트가 인식을 엄청 못해서 에셋 스토어의 API 사용하는데도 인식 때문에 수정 엄청함
- 구버전이라서 gpt도 2.0으로 알려주거나 기능이 없어진 함수를 제공함, 지금은 정상적으로 인식
- 인식을 너무 못해서 관절을 바꿈(엉덩이 중앙?)

7. 배경 (백그라운드) 스크립트
- 배경은 게임 내에서 움직이는것 처럼 보이게하려면 정적인 이미지 파일만 첨부하는 것이 아니라
- 스크립트를 통해 배경이 점점 왼쪽으로 이동하게끔 해야함(어느 특정 위치를 지나게 되면 다시 위치 초기화)
- 시간이 지날수록 이 이동속도가 증가하여 마치 플레이어는 좀 빨라지는 체감이 들게함
- 물론, 장애물이 빨라지는건 아니라서 난이도가 상승하는 것은 아님(이는 장애물 스크립트에서 빠르게해야함)
- 다중 배경(예정, 안할수도있음)

8. 장애물 스크립트
- 처음에는 점프용 장애물만 구현했음
- 나중에 슬라이드를 통해 피하는 장애물을 추가할 예정
- 역시 플레이어가 달리는 것처럼 보이게 하기 위해서 장애물을 왼쪽으로 이동시켜서 플레이어가 이동하는 것처럼 보이게함
- 장애물 속도를 시간이 지나갈수록 빠르게 하여 난이도가 상승 (예정)



9. 아이템
- 전에 코인 패턴을 통해 랜덤으로 나오게 했는데, 그게 장애물이랑 겹쳐서 이상했음
- 그래서 랜덤 생성 기능이랑 풀 관리 기능 뺌
- 코인을 제외한 아이템은 대부분 일정 시간동안 무적 상태 유지
- 코인(먹으면 50점 증가)
- 거대화(먹으면 동일 점수에, 장애물을 파괴시켜줌)

- 특정 시간동안 빨라져서 장애물 파괴 기능 아이템 (예정)
- 장애물을 순간적으로 없애는 아이템 (예정)
 
- 방어막 아이템(예정) - 닿기 전까지는 무적 상태
- 점수 증가 아이템(예정) - 50점이 아닌 80점 정도
- 코인 아이템 생성 주기 늘리기(예정)
- 자석 아이템(예정)

10. 낙하 시스템?
- 무한런에서 생각보다 구현하기 까다로움
- 점점 더 빨라지는 맵에서 낙하 시스템까지 넣으면 난이도 훨씬 증가함
- 같은 배경만 쓰는 스크롤링 과정에서 낙하까지 쓰면 힘들어짐


11. 점수 시스템
- 코인을 먹으면 점수 증가. (50점)
- 캔버스를 통해 점수를 표기하는 방식
- 최고 점수를 기록해서 기록용으로 쓰기 가능

- 장애물 파괴시 점수 올리기(예정)

12. 땅
- 땅 오브젝트는 무한히 반복되는게 아니라, 왼쪽으로 이동하고 벡터 이동으로 위치 초기화해서 땅이 무한으로 반복되게끔 함

13. 게임 오버 판정

- 장애물에 닿으면 즉시 죽음
- 재시작 버튼 구현(재시작 시 키넥트 인식이 아예 안되는 오류가 있음)
- 제스처를 통해 재시작 버튼을 누르는 것(예정)


14. 효과음 및 이펙트
 - 점프 시 효과음
- 슬라이드 시 효과음
- 아이템 먹으면 효과음 및 이펙트, 점수 증가 텍스트 표시(서서히 없어짐)
- 장애물 파괴 시 이펙트 
- 장애물 파괴 시 효과음(예정)


15. 애니메이션
- 아직 아예 안함(안할수도 있음)(예정)
